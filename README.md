# openclaw-eufy-robovac

OpenClaw skill for controlling Eufy RoboVac robot vacuums via the Eufy Home API.

## Features

- Start, stop, and pause cleaning sessions
- Send robot to dock
- Check battery level and cleaning status
- Discover vacuums on the local network

## Contents

- `scripts/eufy-control.py` - Main control script
- `scripts/discover.py` - Local network discovery for Eufy devices
- `references/api.md` - Eufy API documentation
- `references/supported-models.md` - List of tested/supported RoboVac models
- `references/credentials.md` - How to obtain Eufy API credentials

## Setup

See `references/credentials.md` for obtaining your Eufy account credentials.
